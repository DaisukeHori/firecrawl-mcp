11 MCP ツール

スクレイピング・クロール・検索・抽出を完全カバー

firecrawl_scrape

単一URLをMarkdown/HTML/スクリーンショットで取得

firecrawl_batch_scrape

複数URLを並列スクレイピング

firecrawl_batch_cancel

バッチジョブのキャンセル

firecrawl_batch_errors

バッチのエラー詳細取得

firecrawl_crawl

サイト全体をクロール

firecrawl_crawl_status

クロール進捗・結果・キャンセル

firecrawl_crawl_errors

クロールのエラー詳細取得

firecrawl_crawl_active

実行中ジョブ一覧

firecrawl_search

Web検索 + ページ内容取得

firecrawl_map

サイトの全URL一覧取得

firecrawl_extract

LLMで構造化データ（JSON）抽出

クイックスタート

1

Firecrawl を用意

セルフホストガイドに従ってFirecrawlを起動。docker compose up -d で localhost:3002 に起動します。

2

MCP サーバーをデプロイ

上の「Deploy with Vercel」ボタンを押して環境変数を入力するだけ。

FIRECRAWL_API_URL=https://your-firecrawl.example.com
FIRECRAWL_API_KEY=your-api-key
MCP_API_KEY=your-mcp-secret
3

Claude から接続

Claude.ai → Settings → MCP → Add で URL と認証ヘッダーを設定。

URL: https://your-app.vercel.app/api/mcp
Header: x-api-key: your-mcp-secret

今すぐ始める

ワンクリックでデプロイ。セルフホストFirecrawlをClaudeから操作。
